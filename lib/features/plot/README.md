# Plot Feature

## Purpose
Owns plot and real estate inventory workflows for FinHub.

## Responsibilities
- Define plot listing entities.
- Provide plot directory state through Riverpod providers.
- Display available, booked, and sold plots with portfolio metrics.

## What Belongs Here
- Plot-specific entities such as `PlotListing`.
- Plot providers such as `plotProvider`.
- Plot directory pages, widgets, services, use cases, models, and repositories.

## What Should NOT Be Placed Here
- Loan, payment, booking, inventory, dairy, or shared platform logic.
