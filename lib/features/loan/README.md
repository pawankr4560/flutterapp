# Loan Feature

## Purpose
Owns loan management, loan application, document upload, status, and EMI calculator flows.

## Responsibilities
- Present loan pages and widgets.
- Coordinate loan services, request models, entities, and repositories.
- Keep loan-specific data models inside the feature.

## What Belongs Here
- Loan pages, loan widgets, loan services, loan entities, and loan data models.

## What Should NOT Be Placed Here
- Payment checkout models, profile editing services, or generic shared components.
