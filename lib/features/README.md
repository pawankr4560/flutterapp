# Features

## Purpose
Contains independent FinHub business capabilities organized feature-first.

## Responsibilities
- Keep presentation, application, domain, and data layers grouped by feature.
- Isolate feature-specific services, models, repositories, and UI.

## What Belongs Here
- Auth, dashboard, loan, payments, profile, car booking, inventory, dairy, plot, splash, and onboarding modules.

## What Should NOT Be Placed Here
- Cross-feature widgets or models.
- Global app theme or app shell code.
