# Dashboard Feature

## Purpose
Owns the business overview and home navigation experience.

## Responsibilities
- Present dashboard pages, summary cards, and home navigation widgets.
- Coordinate dashboard-specific models and providers.
- Provide the dashboard foundation using mock overview data until real data sources are introduced.

## What Belongs Here
- Dashboard pages, cards, widgets, providers, and dashboard-owned models.
- Dashboard summary entities and presentation-only overview components.

## What Should NOT Be Placed Here
- Loan application services, payment APIs, profile editing, or auth session logic.
