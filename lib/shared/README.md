# Shared

## Purpose
Contains reusable components shared by multiple features.

## Responsibilities
- Store generic UI components, dialogs, animations, and reusable models.
- Avoid coupling shared code to one business flow.

## What Belongs Here
- Reusable components, shared models, common dialogs, and animations.

## What Should NOT Be Placed Here
- Feature-specific DTOs, entities, services, or repositories.
